# Menú de Restaurante en Python 

Sistema simple de menú usando herencia en Python.

##  Archivos
- `Platillo.py` - Clase base con nombre y precio
- `Comida.py` - Comidas con categoría (Principal, Entrada, etc.)
- `Bebida.py` - Bebidas con temperatura (Fría/Caliente)
- `Postre.py` - Postres con indicador sin gluten
- `main.py` - Script de demostración

# Aventureros RPG en Python 

Sistema de personajes de RPG usando herencia en Python.

##  Archivos
- `Aventurero.py` - Clase base con nombre y nivel
- `Guerrero.py` - Ataca con arma cuerpo a cuerpo
- `Mago.py` - Lanza hechizos mágicos
- `Arquero.py` - Dispara flechas (con contador)
- `main.py` - Script de demostración
