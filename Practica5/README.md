# Abstracción en Python 

El archivo abstraccion.py demuestra el concepto de abstracción en programación orientada a objetos usando Python.
Define una clase abstracta Animal con un método abstracto hablar(), y dos clases concretas Perro y Gato que implementan ese método.
Al final, crea instancias de ambas y muestra los sonidos que hacen.

##  Archivo
- `abstraccion.py` - Demostración de abstracción con animales

##  Conceptos
- **Clase abstracta** (`Animal`) que no puede instanciarse directamente
- **Método abstracto** (`hablar`) que obliga a las subclases a implementarlo
- **Polimorfismo** - Cada animal implementa su propio sonido
