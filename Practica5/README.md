# Abstracción en Python 

Ejemplo simple de clase abstracta usando el módulo `abc` de Python.

##  Archivo
- `abstraccion.py` - Demostración de abstracción con animales

##  Conceptos
- **Clase abstracta** (`Animal`) que no puede instanciarse directamente
- **Método abstracto** (`hablar`) que obliga a las subclases a implementarlo
- **Polimorfismo** - Cada animal implementa su propio sonido
