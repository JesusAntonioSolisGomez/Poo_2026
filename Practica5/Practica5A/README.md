#  Práctica 5A - Mobs (POO en Python)

##  Descripción
Se implementa una clase abstracta `Mob` usando `ABC` y `@abstractmethod`, de la cual heredan distintos mobs (`Vaca`, `Creeper`, `Enderman`).  
Cada uno define su propio sonido, comportamiento y forma de moverse.

##  Conceptos
- Clases abstractas
- Herencia
- Polimorfismo

##  Estructura
Practica5A/
├── mob.py
├── Vaca.py
├── Creeper.py
├── Enderman.py
└── main.py

## Nota 
No se puede instanciar Mob directamente porque es una clase abstracta.
