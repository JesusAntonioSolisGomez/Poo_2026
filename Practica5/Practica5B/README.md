#  Práctica 2 - Fábrica de Herramientas (POO en Python)

## 📌 Descripción
Se modela una fábrica de herramientas usando una clase abstracta `Herramienta`.  
Las subclases (`Pico`, `Espada`, `Pala` y `Arco`) implementan su comportamiento específico.

El daño se calcula automáticamente según el material.

##  Conceptos
- Clases abstractas (`ABC`)
- `@property` abstracta
- Herencia
- Polimorfismo
- Encapsulamiento
