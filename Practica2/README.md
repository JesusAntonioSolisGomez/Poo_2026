# Práctica 2 – Cuenta Bancaria y Mascota Virtual

## Descripción

En esta práctica se crean dos clases en **Python** utilizando **Programación Orientada a Objetos**: `cuentaBancaria` y `Mascota`.

## Funcionalidad

**CuentaBancaria**

* Crear una cuenta con titular, número y saldo
* Depositar dinero
* Retirar dinero
* Consultar saldo

**Mascota**

* Crear una mascota con nombre, tipo y edad
* Alimentarla
* Jugar con ella
* Mostrar su estado y nivel de felicidad

## Archivos

* `cuentaBancaria.py`
* `mascotaVirtual.py`
* Diagramas `.drawio`
