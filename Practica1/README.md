# Práctica 1 – Clase Estudiante

## Descripción

En esta práctica se implementa una clase **Estudiante** utilizando **Programación Orientada a Objetos en Python**.
La clase permite almacenar información de un estudiante y gestionar sus calificaciones.

## Funcionalidad

El programa permite:

* Crear objetos de tipo **Estudiante**
* Guardar nombre, edad y carrera
* Agregar calificaciones
* Calcular el promedio del estudiante

## Archivos

* `Estudiante.py` → Implementación de la clase y uso del programa
