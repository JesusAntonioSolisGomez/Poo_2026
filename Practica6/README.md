#  Manejo de Excepciones en Python

Este proyecto contiene ejemplos básicos de cómo manejar errores en Python utilizando bloques `try`, `except`, `else` y `finally`.

##  Archivos incluidos

- `excepcionesA.py` → Manejo de errores en división.
- `excepcionesB.py` → Manejo de errores al acceder a una lista de colores.
- `excepcionesC.py` → Acceso a listas y diccionarios.
- `excepcionesD.py` → Validacion de entrada con bucle.
- `excepcionesE.py` → Leer un archivo con amnejo de errores.
- `excepcionesF.py` → Excepciones personalizadas con clase.
- `reto_calificaciones.py` → Registro de calificaciones.

---

##  Objetivo

## 📖 Explicación general

En esta práctica se estudia el manejo de excepciones en Python, una herramienta fundamental para controlar errores durante la ejecución de un programa. En lugar de que el programa se detenga cuando ocurre un error, se pueden utilizar bloques `try`, `except`, `else` y `finally` para manejar estas situaciones de manera controlada.

Se comienzan abordando errores básicos, como la división entre cero y la entrada de datos incorrectos, utilizando excepciones como `ZeroDivisionError` y `ValueError`. Posteriormente, se trabaja con estructuras de datos como listas y diccionarios, donde se presentan errores como `IndexError` y `KeyError` al intentar acceder a posiciones o claves inválidas.

Además, se implementa una función para validar datos ingresados por el usuario, asegurando que solo se acepten valores correctos mediante el uso de ciclos y manejo de excepciones. También se introduce el manejo de archivos, permitiendo leer y escribir información en archivos `.txt`, controlando errores como la inexistencia del archivo o la falta de permisos.

Finalmente, se desarrollan excepciones personalizadas que permiten definir errores específicos del programa, como edades inválidas o saldo insuficiente. Estas excepciones hacen que el código sea más claro, organizado y fácil de mantener.

En conjunto, la práctica demuestra cómo aplicar el manejo de errores en distintos escenarios, mejorando la seguridad y estabilidad de los programas.
