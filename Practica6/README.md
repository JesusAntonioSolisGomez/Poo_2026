#  Manejo de Excepciones en Python

Este proyecto contiene ejemplos básicos de cómo manejar errores en Python utilizando bloques `try`, `except`, `else` y `finally`.

##  Archivos incluidos

- `excepcionesA.py` → Manejo de errores en división.
- `excepcionesB.py` → Manejo de errores al acceder a una lista de colores.

---

##  Objetivo

Aprender a prevenir fallos en programas mediante el uso de excepciones, controlando errores comunes como:

- Entrada de datos incorrecta  
- División entre cero  
- Índices fuera de rango
