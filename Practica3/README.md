# Proyecto Animales en Python 🐕🐈

Proyecto simple para demostrar **herencia** y **polimorfismo** en Python.

## 📁 Archivos
- `Animal.py` - Clase base con atributos nombre/edad y métodos describir() y hablar()
- `Perro.py` - Clase Perro que hereda de Animal (ladra)
- `Gato.py` - Clase Gato que hereda de Animal (maúlla)
- `main.py` - Script principal que crea objetos y prueba métodos

## ▶️ Ejecutar
```bash
python main.py
